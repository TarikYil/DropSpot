# DropSpot Frontend

Modern React + Vite + Tailwind CSS frontend uygulaması.

## Özellikler

- Modern ve kullanıcı dostu UI
- Sign in / Sign up
- Ana sayfa (Drop listesi)
- Admin Paneli
- Süper Admin Paneli
- Ayarlar sayfası
- AI Chatbot widget (sağ altta)
- Responsive tasarım

## Teknolojiler

- React 18
- Vite
- Tailwind CSS
- React Router
- Axios
- Framer Motion
- Lucide React Icons

## Çalıştırma

```bash
# Development
npm run dev

# Build
npm run build

# Preview
npm run preview
```

## Docker

```bash
docker-compose up frontend
```

## Port

Frontend servisi `http://localhost:3000` adresinde çalışır.
